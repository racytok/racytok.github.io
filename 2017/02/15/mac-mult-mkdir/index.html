Mac上批量给png图片建立对应文件夹

发表于 2017-02-15   |   分类于 偷懒手册   |  
  • 之前写了篇关于windows上根据png批量创建对应文件夹的教程。最近换了mac,所以出一篇mac上操作的教程。
  1. 打开“终端”，切换到放png图片的文件夹
    cd [文件夹地址]
  2. 执行 ls 命令，显示出所有的png图片列表
  3. 新建excel，将列表复制到第一列中
  4. 查找和替换掉.png
  5. 点击第二列第一个单元格 在顶部 fx 后写 “mkdir ”&A1 点击回车，会发现该单元格与前一个单元格相比多了 mkdir （这个就是创建文件夹命令）
  6. 选中刚才那个单元格，按住command和鼠标左键，光标变为+，向下拖动到最后一列。
  7. 完成6后，将第二列内容复制到新建的文档中，在第一列加上，cd [新文件夹存放地址]，保存文档扩展名改为.sh
  8. 在终端中打开 这个.sh文件。此时，就可以看到在 [新文件夹存放地址]里面已经有了我们需要的文件夹

注意：执行第8个步骤的时候，可能会涉及到Permission denied。就是没有权限。
解决办法：
修改该文件xx.sh 的权限 ：使用命令：
chmod 777 xx.sh
然后再执行8，就没问题了。
